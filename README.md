# Secure Login System

## Website URL
Access the website at:
http://localhost/secure_login_system/

## Purpose
This website is a secure login system for user authentication. It allows users to register, log in, and log out securely. User credentials are stored in a database, ensuring only registered users can access protected pages (like the dashboard). The system is designed for learning and demonstration purposes, showing how to implement authentication in PHP with a MySQL database.

## How to Use
1. Start Apache and MySQL in XAMPP.
2. Open your browser and go to: http://localhost/secure_login_system/
3. Create a new account by clicking "Create one" on the login page.
4. Log in with your credentials to access the dashboard.
5. You can log out anytime using the logout button.

## Features
- User registration
- Secure login/logout
- Password hashing
- Session management
- Protected dashboard (only for logged-in users)

---
For first-time setup, visit http://localhost/secure_login_system/install.php to create the database and default user.
